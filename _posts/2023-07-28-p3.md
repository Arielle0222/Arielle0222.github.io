# Code Tree_Input&Output



## #1. OUTPUT

```python
**## 변수 포맷**
a = 5
print("A is %d" % a)

a = 33.567268
print("%.4f" % a)  #%뒤에 .4f : 소숫점 **네번째 자리까지** 반올림

b = "apple"
print("B is %s" % b)

print("A is %d and B is %s" % (a, b))

>>>
A is 5
33.5673
B is apple
A is 5 and B is apple

# %s 문자열
# %c 문자
# %d 정수 
# %f 실수

================================================================================================================
**## format 함수 이용 **
a, b = 5, "apple"

print(f"A is {a}")
print(f"B is {b}")
print(f"A is {a} and B is {b}")

>>>
A is 5
B is apple
A is 5 and B is apple

**## format 함수 이용(소숫점) **

a = 33.567268
print(f"{a:.4f}")  #{a} 에서 a 뒤에 **:.4f 붙이기!**

>>>
33.5673

================================================================================================================
**## 예시1**
a, b = 3, "C"

print(f"3...C")
print("3...C")
print(f"{a}...{b}"). #이걸로 계속 풀자!! 입력값의 순서 바뀌어도 괜춘하다!!
#f는 따음표 **밖에** 있다!
>>>
3...C


**## 예시2**
w, g = 13, 0.165
total  = w*g

print(f"13 * 0.165000 = {total:.6f}")
print(f"{total:.6f}")
**## 두 변수의 값 교환방법1 **
a, b = 5, 3

temp = a #temp = 5가 저장 됨.
a = b #a=3, b=3
b = temp #b=5,,, 즉 최종적으로 a=3, b=5

print(f"A is {a} B is {b}")

>>>
A is 3 B is 5

**## 두 변수의 값 교환방법2**
a, b = 5, 3

a, b = b, a #a = 3, b=5

print(f"A is {a} B is {b}")

>>>
A is 3 B is 5

**##예시**
a, b, c = 1, 2, 3

a = b = c = **sum([a, b, c]) #리스트로 sum 한다!!**

print(f"{a} {b} {c}")

>>>
6 6 6
```
