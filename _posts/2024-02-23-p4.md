# [Code Tree] 단순 반복문 / for 안의 if문 / a부터b까지



[TOC]

##### 🤔 [Question]

***정수 a부터 b까지 숫자를 출력한다. a에서 시작하여 조건에 맞춰 수를 변경하다 b보다 커지는 순간에  종료해야 한다.***

- ***숫자가 홀수인 경우에는 2배가 된다.***
- ***숫자가 짝수인 경우에는 3만큼 증가한다.***

##### 🤨 [Example]

```c++
//INPUT
2 13
  
//OUTPUT
2 5 10 13
```



## 1. Problem

```c++
//ERROR_CODE

#include <iostream>
using namespace std;

int main() {
    int a,b;

    cin >> a >> b;

    for(int i=a ; i<=b; i++){
        if(a%2==0){
            cout << a*2 << " ";
        }
        else if(a%2==1){
            cout << a+3 << " ";
        }
    }
    // 여기에 코드를 작성해주세요.
    return 0;
}
```



## 2. Solution

```c++
//CORRECT_CODE

#include <iostream>
using namespace std;

int main() {
    // 변수 선언
    int a, b;

    // 입력
    cin >> a >> b;

    // 출력
    for (int i = a; i <= b; ) {
        cout << i << " ";
        if (i % 2 == 1)
            i *= 2;
        else
            i += 3;
    }

    return 0;
}

```
