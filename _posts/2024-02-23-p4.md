# [Code Tree] 단순 반복문 / for 안의 if문 / a부터b까지



[TOC]

##### 🤔 [Question]

***정수 a부터 b까지 숫자를 출력한다. a에서 시작하여 조건에 맞춰 수를 변경하다 b보다 커지는 순간에  종료해야 한다.***

- ***숫자가 홀수인 경우에는 2배가 된다.***
- ***숫자가 짝수인 경우에는 3만큼 증가한다.***

##### 🤨 [Example]

```c++
//INPUT
2 13
  
//OUTPUT
2 5 10 13
```



## 1. Problem

```c++
//ERROR_CODE

#include <iostream>
using namespace std;

int main() {
    int a,b;

    cin >> a >> b;

    for(int i=a ; i<=b; i++){
        if(a%2==0){
            cout << a*2 << " ";
        }
        else if(a%2==1){
            cout << a+3 << " ";
        }
    }
    // 여기에 코드를 작성해주세요.
    return 0;
}
```

변수 a,b에 대해 for루프를 사용했다. 문제는 반복문 내 항상 a에 대한 조건을 확인하고 출력하고 있다는 점이다. 즉, **i를 이용하여 조건을 확인하지 않고 항상 a에 대한 조건만을 확인**해 결과값이 일정하게 나오는 것임. 

| 반복 | <u>a</u> |  i   | 출력 값 |
| :--: | :------: | :--: | :-----: |
|  1   |    2     |  2   |    4    |
|  2   |    2     |  3   |    4    |
|  3   |    2     |  4   |    4    |
|  4   |    2     |  5   |    4    |

## 2. Solution

```c++
//CORRECT_CODE

#include <iostream>
using namespace std;

int main() {
    // 변수 선언
    int a, b;

    // 입력
    cin >> a >> b;

    // 출력
    for (int i = a; i <= b; ) {
        cout << i << " ";
        if (i % 2 == 1)
            i *= 2;
        else
            i += 3;
    }

    return 0;
}

```

a부터 b까지의 범위 사이에서 for루프가 실행된다. 이 코드 역시 i를 a로 초기화하지만 루프 내 증가시키지는 않는다. 하지만 **a가 아닌 i를 이용해 홀수,짝수 여부에 따라 값이 업데이트** 된다. 

| 반복 |  a   | <u>i</u> | 출력 값 |
| :--: | :--: | :------: | :-----: |
|  1   |  2   |    2     |    2    |
|  2   |  2   |    5     |    5    |
|  3   |  2   |    10    |   10    |
|  4   |  2   |    13    |   13    |
